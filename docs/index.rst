Noteslib documentation
====================================
``noteslib`` is a library of Python classes for manipulating Notes/Domino objects via COM.

``noteslib`` was created by Robert Follek, and the current maintainer is Marcelo Huerta.

The ``noteslib`` classes correspond to the standard LotusScript classes; they support all the standard properties
and methods. The ``noteslib`` classes have additional methods and ease-of-use features.

See below the details for the individual classes.
